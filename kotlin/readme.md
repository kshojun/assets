# 特徴
- コードが簡潔
- 文末のセミコロン不要
- Android開発公式言語
- 変数はNullableとNotNullに分かれ、?をつけてNullableを宣言可能
- NullPointerException(NPE)から自由
- 関数型プログラミングとOOP両方可能

# 使用可能プラットフォーム
- Kotlin/JVM 
- Kotlin/JS 
- Kotlin/Native

# Kotlin/Nativeのターゲット
- iOS
- MacOS
- Android
- Windows
- Linux
- WebAssembly

# JDK
- Oracle JDK 一部ライセンス料必要
- OpenJDK Oracle JDKとほぼ同じ
- AzulのZulu(OpenJDKを配布するサードパーティー)
