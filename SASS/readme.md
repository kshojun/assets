拡張子はscss   

css
```html
.container h1 {
    color: red;
}

.container p {
    color: blue;
}
```

scss
```html
.container {
  h1 {
    color: red;
  }
  p {
    color: blue;
  }
}
```

# Install VSCode Extensions
- Live Sass Compiler
- Beutify css/sass/scss/less

# Using Live Sass Compiler
scss fileをクリックし、Watch Sassをクリックするとmapファイルとcssファイルができあがる

# Using Beutify css/sass/scss/less
Shift+Ctrl+Pでbeutifyを選択
