# n
```bash
# Confirm node.js version
$ node -v
# Installation of n
$ npm install n -g
# Confirm node.js version
$ n --stable
$ n --latest
# Installation of node.js
$ n latest or n stable
# Switch node.js version
$ n
# List of installed node.js
$ n ls
```
